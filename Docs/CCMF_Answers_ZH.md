## A. 摘要 / 項目簡介 (150-200 字)

Sightline 透過混合 AR 錨定技術，為香港文化遺產地標提供實物上的即時解答。用戶將智能手機指向地標——鐘樓、天星小輪、星光大道——系統在 2 秒內將信息卡錨定在實際物體上，並顯示可見的引導線。

**問題：** 遊客和學生難以投入文化遺產景點。傳統語音導覽缺乏空間脈絡；現有 AR 應用需要複雜設置或顯示通用 HUD 覆蓋。

**解決方案：** 零設置 AR 結合基於 GPS 的 ARCore Geospatial（戶外、快速）和視覺 AI（室內、精確）。系統適應環境——開放區域使用 GPS，室內使用相機識別——確保跨場景可靠錨定。

**創新：** 混合錨定策略配備信心分級（綠/琥珀/灰狀態）、Quest 3 Passthrough Camera API 整合、<2 秒響應時間。面向未來的架構支持 AR 眼鏡。

**市場：** 香港博物館、文化遺產場館、教育機構，透過 B2B2C 場館 SDK 模式。

**社會影響：** 500+ 香港學校（35 萬學生）免費使用；透過沉浸式學習保存文化知識；將學生在文化遺產景點的參與度提高 20-30%。

**6 個月目標：** 20 個 POI 上線、50 個測試用戶、1 個博物館試點、第 4 個月公開發布。

## D. 創造力和創新 (30%)

### 創新技術

**1. 混合 AR 錨定（核心創新）**

傳統 AR 應用使用單一方法錨定（僅 GPS 或僅圖像）。Sightline 根據情境動態選擇最佳方法：

- **戶外（基於 GPS）：** ARCore Geospatial API
  - 延遲：p50 <0.6 秒（GPS 鎖定 + API 調用 + 錨點放置）
  - 準確度：±5-15 米（對於 >10 米的地標足夠）
  - 覆蓋範圍：任何有 GPS 信號的戶外位置

- **室內（基於視覺）：** Quest 3 Passthrough Camera API + Gemini Vision AI
  - 延遲：p50 <1.5 秒（幀捕獲 + AI 推理 + 2D→3D 轉換）
  - 準確度：±0.5 米（光線投射到檢測到的物體表面）
  - 覆蓋範圍：任何光線充足、具有明顯視覺特徵的室內環境

- **備援（高精度）：** Vuforia 區域目標（可選）
  - 延遲：p50 <1.2 秒（圖像識別 + 姿態估計）
  - 準確度：±5 厘米（預掃描環境）
  - 覆蓋範圍：博物館、畫廊（需要一次性掃描）

**技術差異化：**
- 信心狀態機：綠色（≥0.85）/ 琥珀色（0.60-0.85）/ 灰色（<0.60）
- 優雅降級：如果 GPS 漂移，系統提示 "重新瞄準相機以鎖定"
- 跨設備一致性：手機、Quest 3、未來眼鏡上的相同 POI 數據

**2. 最先進的平台 API**

- **Meta Quest 3 Passthrough Camera API**（實驗性，OS v74+）
  - 用於 MR 體驗的直接相機訪問
  - 通過環境光線投射管理器進行 2D→3D 轉換
  - 無需預掃描即可實現室內 POI 檢測

- **Unity Sentis**（設備上 ML）
  - 在雲 API 調用之前進行本地圖像預處理
  - 減少帶寬並改善隱私

- **ARCore Geospatial API**（Google）
  - VPS（視覺定位系統）在特定區域提供厘米級戶外精度
  - Streetscape Geometry 用於引導線邊緣貼合

**3. 性能突破：<2 秒首次價值時間**

行業平均 AR 內容加載時間：5-8 秒（Niantic Lightship、博物館 AR 應用）。

Sightline 通過以下方式實現 p50 ≤2.0 秒：
- 並行處理：GPS + API 調用 + 錨點設置在並發線程中
- 響應緩存：50 米網格瓦片緩存 POI 數據（10 分鐘 TTL）
- 預測性預取：當用戶在 100 米範圍內時加載附近的 POI
- 優化資產管線：卡片渲染為輕量級廣告牌（每個 <50KB）

**可測量的 KPI：**
- GPS 路徑：p50 延遲 0.4-0.6 秒（在鐘樓測試）
- 視覺路徑：p50 延遲 1.2-1.5 秒（使用海報模型在室內測試）
- 錨點穩定性：行走 3-4 米時 ≥10 秒連續鎖定

### 創意解決方案

**1. 引導線可視化**

**問題：** 用戶忘記 AR 卡片指的是哪個物體（遮擋、混亂）。

**解決方案：** 從錨點到卡片的動態 SVG 線，具有：
- 邊緣貼合：線條遵循建築物邊緣（使用 Streetscape Geometry）
- 深度感知：當物體在用戶身後時線條褪色
- 視覺層次結構：較粗的線條 = 更高的信心鎖定

**2. 遠場方位廣告牌**

**問題：** GPS 精度在 100 米以外降低；無法錨定到遠處的地標（IFC @ 2 公里）。

**解決方案：** 帶有距離芯片的基於方位的廣告牌：
- 從用戶姿態到地標計算方位/仰角
- 在該方向渲染卡片（不是世界鎖定）
- 顯示 "IFC · 2.1 公里" 以設定期望
- 當用戶接近 <100 米時切換到本地錨定

**3. 信心分級 UI**

**問題：** 二進制鎖定/失敗會造成挫敗感（閃爍、誤報）。

**解決方案：** 具有清晰用戶反饋的三狀態系統：
- **綠色：** 鎖定，顯示完整卡片 + 啟用交互
- **琥珀色：** 不確定，顯示 2 個最近的 POI 芯片，提示 "哪一個？"
- **灰色：** 搜索中，顯示 "穩定相機..." 提示

### 顛覆性能力

**顛覆傳統博物館語音導覽：**
- 語音導覽：被動，需要手動選擇（"按 7 選擇展品"）
- Sightline：主動，自動識別（"查看展品 → 即時信息"）

**顛覆現有 AR 文化遺產應用：**
- 競爭對手需要特定應用設置、預掃描或僅 GPS（室內失效）
- Sightline：零設置，室內/戶外工作，適應環境

**市場影響：**
- 將博物館入職摩擦從 5 分鐘（下載應用、選擇語音導覽）減少到 <30 秒（掃描二維碼、指向手機）
- 實現 "即走即用" 體驗：鐘樓遊客無需事先計劃即可立即獲取信息
